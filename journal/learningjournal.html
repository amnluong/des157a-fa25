Modal/ Overlay/ Dialog Window

10.22.25

A modal window is an element that sits on top of an application’s main window. It disables the main window but keeps it visible with the model window as a child window in front of it. You may want to use the modal window when you need to grab the user’s attention, need user input, show additional information in context, and show additional information that is not in context...
