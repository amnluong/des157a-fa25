Photos With Visual Thinking Strategies

10.26.25

Pictures tell us everything from the small details to big details. I think we have to become detectives when we look at a picture because there are a lot of details hiding from the main event. It takes a lot of practice observing pictures and predicting what is happening in the photos. I would need to keep on asking myself open-ended questions because I wouldn't be able to think about something unless I was asked a question. Someone asking me questions about a photo, will allow me to notice more details. Looking at images from around the world opens my eyes to other cultures and how other people live their lives. It is interesting to see how different one life can be from my own life.

Imagary and Interaction to Inspire Visual Thinking

10.26.25

This website uses imagery to create interesting visuals as you go through the website. You can interact with websites by scrolling through or moving your mouse to create motion. The strengths of the website is how interactive the website is so it does not get boring. There is a lot of imagery so everything you see as you scroll is new. Each imagery is for each topic so you can think about the meaning of each image that fits within a certain content. A drawback may be how overstimulating the website may be. There is a lot going on with a lot of images everywhere, so people may find it too much.
